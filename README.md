# Employee CRUD Backend

This is a simple backend project built using Spring Boot for managing employee data.

In this project, I created REST APIs to perform basic CRUD operations like add, update, delete and view employees.

## Tech Stack
- Java
- Spring Boot
- JPA / Hibernate
- MySQL

## Features
- Add new employee
- Update employee details
- Delete employee
- Get all employees

## API Endpoints
- GET /show_all_emps
- POST /save
- PUT /update
- DELETE /delete/{id}

## How to run
Open the project in Eclipse and run the Spring Boot application.

## Note
This project helped me understand how backend APIs work and how they connect with frontend applications.
